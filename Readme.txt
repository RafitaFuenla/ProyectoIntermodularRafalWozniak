# BoxRanger – Gestión de gimnasio CrossFit

## Descripción
BoxRanger es una aplicación pensada para gestionar un gimnasio de CrossFit.

La idea del proyecto es poder llevar el control de socios, entrenadores, clases, inscripciones y pagos en un mismo sistema. Este proyecto forma parte del intermodular de 1º de DAM, donde se busca unir lo aprendido en varias asignaturas en una aplicación real.

---

## Objetivo
El objetivo es simular el funcionamiento de un gimnasio real, permitiendo gestionar la información básica del día a día:

- Alta y gestión de socios  
- Organización de clases  
- Control de pagos  
- Inscripciones a clases  

---

## Tecnologías utilizadas
- Java  
- MySQL  
- JDBC  
- Git y GitHub  
- XML y XSD

---

## Funcionalidades principales
- Gestión de socios  
- Gestión de entrenadores  
- Creación y consulta de clases  
- Inscripción de socios en clases  
- Control de pagos  

---

## Ejecución
Para ejecutar el proyecto:

1. Crear la base de datos con el script SQL  
2. Ejecutar la aplicación en Java  
3. Conectar la aplicación a la base de datos mediante JDBC
